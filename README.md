#Kipsum

[Lorem Ipsum](https://ko.wikipedia.org/wiki/로렘_입숨)과 같은 무의미한 텍스트를 한글로 생성해줍니다.

[Thefron](http://thefron.me)님의 [한글입숨](http://hangul.thefron.me/)을 이용하였습니다.


#설치

```
$ go get github.com/minsoopark/kipsum
```


#사용법

```
$ kipsum {문단 수} {문단의 길이}
```

- 문단 수 : 1~20의 숫자
- 문단의 길이 : `short`, `medium`, `long`
